# Source Effectiveness Tracking

## Overview
Tracking performance of different news sources for economic news coverage. This data informs source selection strategies and helps optimize search effectiveness.

## Source Evaluation Criteria

### 1. Success Rate (40%)
**Definition**: Percentage of searches using this source that yield suitable articles
**Measurement**: (Successful searches / Total searches) × 100

### 2. Quality Score (30%)
**Definition**: Average educational value of articles from this source
**Components**:
- IB syllabus relevance: 10%
- Clarity for students: 10%
- Depth of economic analysis: 10%

### 3. Geographic Coverage (20%)
**Definition**: Diversity of geographic regions covered
**Measurement**: Number of distinct regions covered / Total articles

### 4. Timeliness (10%)
**Definition**: Speed of article availability after events
**Measurement**: Average hours between event and article publication

## Source Selection Strategy

### Primary Selection Criteria
1. **Topic Relevance**: Match source strengths to topic
2. **Geographic Perspective**: Ensure diverse viewpoints
3. **Educational Value**: Prioritize clarity and explanation
4. **Timeliness**: Use fastest sources for breaking news

### Source Rotation Rules
1. **Weekly Rotation**: Don't use same source two weeks in a row
2. **Geographic Balance**: Ensure global perspective over month
3. **Topic-Source Matching**: Use best sources for each topic type
4. **Quality Prioritization**: Favor higher-quality sources when available

### Fallback Strategy
1. **Primary Failure**: Try next best source for topic
2. **Multiple Failures**: Broaden search to all reputable sources
3. **Emergency**: Use pre-vetted educational resources

## Source Quality Assessment

### Quality Indicators (Positive)
1. **Clear Explanations**: Economic concepts explained simply
2. **Educational Context**: Connections to broader economic principles
3. **Data Presentation**: Charts, graphs, statistics with explanations
4. **Expert Commentary**: Quotes from economists or policymakers
5. **Historical Context**: Comparison to past events or trends

### Red Flags (Negative)
1. **Excessive Jargon**: Unexplained technical terms
2. **Political Bias**: Clear partisan slant without balance
3. **Sensationalism**: Clickbait headlines over substance
4. **Lack of Depth**: Superficial treatment of complex topics
5. **Outdated Information**: Not current or relevant

## Source List Maintenance

### Regular Updates
1. **Monthly**: Review source performance data
2. **Quarterly**: Assess need for new sources
3. **Annually**: Comprehensive source list review

### Source Addition Criteria
New sources must meet:
1. **Reputation**: Established, credible news organization
2. **Quality**: Meets educational standards
3. **Coverage**: Adds geographic or topical diversity
4. **Accessibility**: Available through search APIs

### Source Removal Criteria
Sources may be removed if:
1. **Consistent Low Quality**: Repeated quality failures
2. **Access Issues**: Technical problems accessing content
3. **Bias Concerns**: Unacceptable political or ideological bias
4. **Redundancy**: Duplicates better-performing sources

## Integration with Search Strategy

### Search Priority by Source
1. **First Pass**: Top 3 sources for topic based on historical performance
2. **Second Pass**: Remaining primary sources
3. **Third Pass**: Secondary sources
4. **Final Pass**: Broad search across all reputable sources

### Search Optimization
1. **Source-Specific Queries**: Tailor search terms to source strengths
2. **Geographic Filters**: Use when seeking regional perspectives
3. **Time Optimization**: Match search timing to source publication patterns
4. **Content Type**: Prefer news articles over opinion pieces
