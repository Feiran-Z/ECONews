# Content Patterns and Best Practices

## Blog Post Structure Template

### Title Format
- `"[YYYY-MM-DD] News Source: News Title"`
- Always use square brackets for date
- Colon separates source from title
- Date format: YYYY-MM-DD for consistency

### Required Elements
1. **Title**: Follows format above
2. **Date**: Human-readable format (e.g., "January 10, 2024")
3. **Source**: Reputable news organization
4. **Article Link**: Placeholder URL format matching source
5. **Summary**: One paragraph (3-5 sentences) covering key points
6. **Analysis**: One paragraph connecting to IB Economics theories (3-5 specific theories/frameworks)
7. **Tags**: Up to 3 relevant economic topics

## IB Economics Integration Framework

### Theory Connection Patterns
1. **Macroeconomic News** → Monetary policy, fiscal policy, inflation, unemployment, economic growth
2. **International News** → Exchange rates, trade, protectionism, globalization, development
3. **Market News** → Supply/demand, market structures, elasticity, market failure
4. **Policy News** → Government intervention, equity vs efficiency, externalities
5. **Development News** → Economic growth, inequality, sustainability, human capital

### Analysis Structure
- Start with "This article/news connects to IB Economics topics on..."
- List 3-5 specific theories/concepts with brief explanations
- Include practical application: "Students can analyze..."
- End with evaluation framework or discussion questions

## Content Quality Standards

### Summary Requirements
- Concise (3-5 sentences)
- Covers main economic event/policy/issue
- Objective and factual
- Avoids opinion/editorial content

### Analysis Requirements
- Specific IB theory references
- Clear connections to syllabus
- Practical application guidance
- Balanced perspective on economic debates

### Tag Selection
- Use standard economic terminology
- Maximum 3 tags per post
- Include both broad and specific topics
- Consistent with IB Economics vocabulary

## SEO and Accessibility

### Metadata Standards
- Use economic keywords naturally
- Include IB Economics as primary keyword
- Structure content with clear headings
- Use semantic HTML tags

### Readability Standards
- Paragraphs: 2-3 sentences maximum
- Sentence length: 15-20 words average
- Active voice preferred
- Define economic terms in context

## Content Update Schedule

### Weekly Analysis Pattern
- 1-2 new posts per week
- Diverse topic coverage
- Balance between micro and macro
- Global perspective maintained

### Topic Rotation
- Week 1: Macroeconomic policy
- Week 2: International economics
- Week 3: Market economics
- Week 4: Development economics
- Week 5: Environmental economics
- Repeat with current events
